Frequently Asked Questions (FAQs)
---------------------------------

# Cloning a repository
  
  - The ``git@code.jlab.org`` in the ``git://git@code.jlab.org/...`` URL is the
    correct username@host for the the SSH git: protocol.  Authentication (when
    needed) is done via ssh-key exchange.  You must upload a public-key to
    your account.
  - See `Clone a Git repository to your local computer <https://docs.gitlab.com/ee/topics/git/clone.html>`_ for details
