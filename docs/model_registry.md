## GitLab Model Registry

[MLflow](https://mlflow.org/docs/latest/ml/) is a popular open source tool for Machine Learning experiment tracking. GitLab [Model experiment tracking](https://docs.gitlab.com/user/project/ml/experiment_tracking/) and GitLab [Model registry](https://docs.gitlab.com/user/project/ml/model_registry/) are compatible with the MLflow client. The setup requires minimal changes to existing code.
